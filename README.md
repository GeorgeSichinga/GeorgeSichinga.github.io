# georgesichinga.github.io

Personal portfolio website for George Sichinga — Applied Data Scientist and Agricultural Economist based in Lilongwe, Malawi.

## About

This site serves as a living CV and project showcase, built as a single HTML file with no frameworks or build steps required. It covers my academic background, research projects, field experience, and technical skills.

## Stack

- Vanilla HTML, CSS, and JavaScript
- Google Fonts — Syne, DM Mono, Instrument Serif
- Simple Icons CDN for technology logos
- No dependencies, no npm, no build process

## Features

- Responsive layout — desktop and mobile
- Animated hero with ambient glow and parallax effect
- Scrolling ticker and section reveal animations
- Active navigation highlighting on scroll
- Technology tags with official icons

## Structure

```
georgesichinga.github.io/
├── index.html          # Entire site — all styles and scripts inline
└── George profile.jpg  # Profile photo referenced by index.html
```

## Deployment

Hosted via GitHub Pages from the root of this repository. Any push to `main` updates the live site automatically.
